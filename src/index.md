## Gremlins?

#### 1. Include GremlinJS

``` html
<script src="js/gremlinjs.min.js"></script>    
```
#### 2. Add a Gremlin

``` html
<div data-gremlin="Gizmo">
  <p> Loading images... </p>
</div>
```
