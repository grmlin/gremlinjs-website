GremlinJS web components the easy way

GremlinJS is a dependency-free library to build reusable web components with Javascript.

Gremlins are modular and flexible pieces of code describing the behaviour of dom elements. Don't waste your time managing dozens of dom queries, let GremlinJS handle that for you.

  • no dependencies
  • fast and lightweight
  • lazy loading of javascript components
  • highly extendable with build in extensions interface
Download GremlinJS ~4k minified and gzipped

Quick Start

1. Include GremlinJS

<!DOCTYPE html>
<html>
  <head>
  </head>
  <body>
    <script src="js/gremlinjs.min.js"></script>
  </body>
</html>

2. Add a gremlin

<body>
    <div class="gizmo-slideshow" data-gremlin="Gizmo">
        <p>
            <strong>Hello World, Gizmo here! Loading some images from google now...</strong>
        </p>
    </div>
    <script src="js/gremlinjs.min.js"></script>   
    <script src="js/Gizmo.js"></script>
</body>
