GremlinJS web components the easy way

GremlinJS is a framework to build reusable components for your websites with Javascript. Gremlins are modular and flexible pieces of code describing the behaviour of dom elements.

  • it has no dependencies
  • it's fast and lightweight
  • it supports lazy loading of javascript components
  • it's highly extendable with build in extension interface

Download GremlinJS 4k minified and gzipped

Quick Start

1. Include GremlinJS

Legacy Support!

GremlinJS uses the javascript JSON API to parse JSON strings from dom element attributes. Include a polyfill like json2.js if you have to support older browsers (e.g. IE7) 
<body>
    ...
    <script src="js/gremlinjs.min.js"></script>
</body>

2. Add a gremlin

<body>
    ...
    <div data-gremlin="Gizmo"></div>
    ...
    <script src="js/gremlinjs.min.js"></script>   
    <script src="js/gremlins/Gizmo.js"></script>
</body>

Gizmo.js

var Gizmo = GremlinJS.define('Gizmo',
    function () {
        this.el.innerHTML = 'Hello World, Gizmo here!';
    }
);
